# 1. 호스팅 알아보기!

## ⓐ 호스팅이란?
- 대형 서버의 기능을 빌려쓰는 것을 의미합니다. <br>
예를 들어 개인 홈페이지를 만들면 이 홈페이지가 들어설 웹 공간이 필요합니다. 일상에서는 부동산이 이에 해당합니다.

- 개인 컴퓨터로 이러한 웹 공간을 마련하고 24시간 가동과 더불어 전 세계인들의 방문을 감당하는 것은 어렵습니다. <br>
그래서 전문 호스팅사의 서버를 빌려 그 안에 나만의 공간을 빌리는 것을 호스팅이라고 합니다.

## ⓑ 호스팅의 종류

- 호스팅사마다 호스팅의 종류는 다양합니다 그 중 대표적인 것으로 웹 호스팅, 서버 호스팅, 클라우드 호스팅으로 구분할 수 있습니다.

**㉠ 웹호스팅**

<img src="https://mblogthumb-phinf.pstatic.net/MjAxODA5MjVfMjg0/MDAxNTM3ODYxMjQ5NzEx.OnbwMGR6QTxMMWQkoU9owIsDDbe4DwpnrhzVbdeQolQg.6xWmh0-bOcMGGyc5aj4fmbEPFCRSPBw_yjlQdadH1Ksg.JPEG.lhhoo1717/24975.jpg?type=w800" width="450px" height="200px">

- 웹 호스팅은 하나의 서버장비를 여러명이 공유하여 사용하는 것을 의미하며, 가격이 저렴하고, 대중적으로 가장 많이 사용하고 있습니다.<br>

- 기업이나 개인 홈페이지는 월트래픽양이 많지 않아서 웹 호스팅을 사용하는 것이 가장 적합합니다.<br>
하지만 트래픽양이 증가해서 혼자 너무 많은 트래픽을 잡아 먹게되면 서버 다운 등의 제약이 생길 수도 있습니다.

**㉡ 서버 호스팅**

<img src="https://mblogthumb-phinf.pstatic.net/MjAxODA5MjVfOTIg/MDAxNTM3ODYxMzc0NjIw.KymdmpEjngt4gpmTN0jxv-wKiXKk4wzcJ_IKYDzTZGcg.YJzfx2OWow7bjymcbDNlD9HxKbFIWvpR4gAcOwQ-Hi4g.JPEG.lhhoo1717/213.jpg?type=w800" width="450px" height="200px">

- 서버 호스팅은 웹 호스팅과 반대로 한명의 고객이 하나의 서버장비를 임대하는 호스팅을 의미하며 가격이 비싸고 스케일과 트래픽양이 많은 대형 홈페이지를 구축할 때 사용하는 서비스입니다.

- 서버를 단독 사용하므로 설치, 삭제 등의 개발이 자유로우며 고정적으로 트래픽양이 많은 사이트에 적합한 호스팅입니다.

**㉢ 클라우드 호스팅**

<img src="https://mblogthumb-phinf.pstatic.net/MjAxODA5MjVfMTAg/MDAxNTM3ODYwOTU1OTA3.CCCO3Dus4b0c3o8LYEChJPpUhNxZSfSpWasEJjxzo-Yg.rVPhJBS8o4xZP5_pKiRmwgTlZIywV4pBPQ2bnh2Ejckg.JPEG.lhhoo1717/277.jpg?type=w800" width="450px" height="200px">

- 클라우드 호스팅은 서버 호스팅과 유사하지만 물리적 서버장비가 아닌 가상서버를 임대한다는 것에서 차이가 존재합니다.

- 자유롭게 서버 스펙을 조절할 수 있고, 이용한 만큼만 금액을 지불하면 된다는 장점이 있으며 일시적인 트래픽 변동량이 많은 사이트에 적합한 호스팅입니다.
